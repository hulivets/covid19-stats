## RUN DEVELOPMENT VERSION
1. `npm i`
2. `cp ./public/static/config.sample ./public/static/config.js`
3. `npm start`
4. open http://localhost:3000 in browser


## BUILD PRODUCTION VERSION
1. `npm i`
2. `cp ./public/static/config.sample ./public/static/config.js`
3. `npm run build`

## TESTS
1. `npm run test`
